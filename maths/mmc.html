Modular Multiplication Circles

This is a desmos-built interactive demonstration of the concept covered in this Mathologer video:

What's going on?

  • Take $n$ evenly-spaced points on a circle, and label them going round starting with $0, 1, 2, \dots, n-1$.
  • For the point labelled $k$, connect it with a straight line to the point labelled $m\cdot k$, where this multiplication is done modulo $n$.
  • Repeating this for all the $n$ points, some interesting patterns emerge...
